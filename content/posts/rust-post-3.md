## 변수와 불변성

Rust에서 변수는 기본적으로 불변(immutable)입니다.

```rust
let x = 5;
// x = 6; // 에러! 불변 변수는 재할당할 수 없습니다.
```

## 가변 변수

변수를 가변으로 만들려면 `mut` 키워드를 사용합니다.

```rust
let mut x = 5;
x = 6; // 정상 작동
```

## 기본 타입들

Rust의 기본 스칼라 타입들을 살펴보겠습니다.

### 정수 타입
- `i8`, `i16`, `i32`, `i64`, `i128` (부호 있는 정수)
- `u8`, `u16`, `u32`, `u64`, `u128` (부호 없는 정수)

### 부동소수점 타입
- `f32`, `f64` (32비트, 64비트 부동소수점)

### 불린 타입
- `bool` (true 또는 false)

### 문자 타입
- `char` (4바이트 유니코드 스칼라 값)
